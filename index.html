Ourmap setup

The app source lives in travel-pins/client and is built with Vite + React. Opening the raw source folder directly in a browser will show a blank page because the TypeScript/JSX entrypoint needs a dev server or a production build.

Use one of the options below to run the app:

  • Local dev (recommended) 
    cd travel-pins/client
npm install
npm run dev
    Then open the URL printed by Vite (usually http://localhost:5173).
  • Production build + API server 
    cd travel-pins
npm install
npm run build
npm run dev:server
    The API server serves the built client from client/dist.

If you still want to peek at the source HTML, you can open travel-pins/client/ directly (it will remain blank without a build or dev server).
